## TTkit
A Node Command Line Tool to auto generate a web structure.

## Install
```bash
npm install ttkit -g
```


## Usage
```bash
ttkit <project name>
```

Then
```bash
cd <project name>
npm install
gulp watch
```

Now you can develop your app...

## Options
```bash
 Usage: ttkit [options] [project name]

  Options:

    -h, --help                   output usage information
    -V, --version                output the version number
```
